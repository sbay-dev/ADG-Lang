# Repository Policy

This public repository is the official public language repository for ADG-Lang.

## Included

- Language rule table.
- Academic reading key and Mermaid maps.
- Public AST examples.
- Public reference compiler release.
- Verification scripts.
- Public contribution guidance.

## Excluded

- IDE state.
- Build intermediates.
- Native binaries generated during local tests.
- Private development repository history.
- Session-local notes that are not public language documentation.

## Release Rule

Public releases should include source, examples, tests, and documentation. They should not commit generated native binaries or local build folders.
