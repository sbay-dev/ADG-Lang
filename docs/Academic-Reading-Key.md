# Academic Reading Key

هذه الوثيقة تثبت مفتاح قراءة النسخة الأكاديمية من ADG-Lang، بحيث يستطيع الباحث أو المساهم أن يفرّق بين النص المنقول، والباب المنسوب، والاستنباط التشغيلي، وحدود النسبة.

## المفتاح المختصر

| اللون | الدلالة | الوصف |
| --- | --- | --- |
| الأخضر | نصوص منقولة بلفظها أو قريبًا منه | مثل تقسيم الكلام وتعريف الاسم والفعل والحرف وتقسيم الظاهر والمضمر. |
| الأزرق | أبواب منسوبة | مثل الفاعل، المفعول، المضاف، الرفع، النصب، الجر، الجزم. |
| البرتقالي | نظام النقط والضبط | مثل نقطة فوق الحرف للفتح، أمامه للضم، تحته للكسر، ونقطتان للغنة/التنوين. |
| البنفسجي | استنباطات تشغيلية | مفيدة للتحليل الحاسوبي، لكنها ليست نصوصًا مباشرة من أبي الأسود. |
| الرمادي | حدود النسبة | ما لا ينبغي إسناده تفصيليًا إليه؛ لأن المصادر تذكر أنه أسس رسومًا ثم جاء من بعده من زاد وفرّع. |

## Proof Degrees

| درجة الثبوت | الاستخدام في ADG-Lang |
| --- | --- |
| نص منقول | يقابل `Textual` في جدول القواعد. |
| باب منسوب | يقابل `Attributed chapter`. |
| استنباط مباشر | يقابل `Operational inference`. |
| نظام ضبط | يقابل `Notation`. |
| تصميم ADG حديث | قاعدة تنفيذية حديثة لا تُنسب تاريخيًا. |
| غير منسوب / حد نسبة | يقابل `Guardrail`. |

## Usage

1. لا تُقبل قاعدة عامة دون تحديد لون/طبقة الدليل.
2. لا تُنسب القاعدة التشغيلية الحديثة إلى أبي الأسود مباشرة.
3. كل قاعدة مرشحة للتنفيذ يجب أن تربط بين: الدليل، والعقد النحوي، ومثال AST.
4. أي خلاف بحثي يجب أن يُحل بتعديل درجة الثبوت لا بإخفائها.
